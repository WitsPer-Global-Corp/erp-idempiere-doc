＃ 目的

本網站的目的是幫助您學習，配置，增強，部署，擴展和支持IDEMPIERE實例以及整個IDEMPIERE社區。

＃idempiere簡介

成立於2011年的Idempiere是一個功能強大的II，開源ERP/CRM/SCM系統，由熟練的社區支持。該項目側重於高質量軟件，一種開放性的理念及其協作社區，其中包括主題專家，集成商，開發人員和最終用戶。

＃為什麼idempiere
- 很容易安裝。
- 它在小服務器上運行。
- 它擴展到數百個用戶。
- 它使用主要且常用的技術。
- 它具有企業質量會計系統。
- 它支持單個登錄名中的多個實體（集書集）。
- 它通常在世界範圍內使用。
- 它得到了一個大型社區的支持。
